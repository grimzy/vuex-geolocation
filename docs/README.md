# vuex-geolocation
  > A vuex plugin to seamlessly integrate the geo-location service

## Features
  - Integrates directly with vuex
  - Lightweight
  - Configurable


## Dependencies
  - [Vue.js 2x](https://vuejs.org/)
  - [Vuex](https://vuex.vuejs.org/en/)
